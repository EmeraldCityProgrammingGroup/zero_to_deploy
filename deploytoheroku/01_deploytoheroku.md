# Heroku #
## Service for hosting Rails applications ##
## Built on top of Amazon EC2 ##
## Free for small applications ##

!SLIDE
# Heroku Set Up #
    $ gem install heroku
    $ heroku keys:add
    $ heroku create
    
!SLIDE commandline
# Deploy to Heroku #
## Execute the following commands: ##
    $ git push heroku master 
    $ heroku open
